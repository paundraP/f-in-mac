# F in Mac

Its been long time I am into the Cyber Security field, especially in Digital Forensic. And the big challenge is, there is many things to analyze is based on Windows, and because of that, there is a lot of tools build only for Windows. I know its not a wrong thing, but I just can't fit. So that's why I make this repository, to gather all of the tools that I hardly find can compatible with MacOS so I rewrite it my own.

Yea, I rewrite it, because most of it already exist but I can't compile or there is no source for Mac user. Big shoutout for anyone who make the tools and open-source it. Here is the tools I implement so it can compatible with my own daily laptops:

- [prefetch-parser](prefetch-parser/) - A tool to parsing a prefetch file that usable for analyzing Windows Artifact, thanks for Eric Zimmerman for creating the OG tools, [PECmd](https://github.com/ericzimmerman/pecmd).

- [srum-dump](srum-dump/) - A tool to parsing a System Resource Utilization Management (SRUM) database. thanks to Mark Baggett for creating the OG tools, [srum-dump](https://github.com/MarkBaggett/srum-dump/tree/master)
